# sorts
## Теоретический анализ алгоритмов
Bogo - факториальная
- Best: O(1)
- Average: O(n!)
- Worst : O(n!)

Selection Sort - экспоненциальная
- Best: O(n ^ 2)
- Average: O(n ^ 2)
- Worst : O(n ^ 2)

Bubble Sort - полиминальная
- Best: O(n)
- Average: O(n ^ 2)
- Worst : O(n ^ 2)

Heapsort - линейный логарифм
- Best: O(n log(n))
- Average: O(n log(n))
- Worst : O(n log(n))

Перебор масива - линейная
- O(n)

Binary Search - логарифмическая
- Best: O(1)
- Average: O(log(n))
- Worst : O(log(n))
